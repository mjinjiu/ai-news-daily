# AI前线 - AI新闻自动聚合网站

⚡ 每日自动更新的人工智能资讯聚合网站

## 🌐 在线访问

- **GitHub Pages**: https://你的用户名.github.io/ai-news-daily
- **Vercel**: https://ai-news-daily.vercel.app

## 📁 文件结构

```
ai-news-site/
├── index.html      # 主页面（自动更新）
├── style.css       # 样式文件
├── app.js          # 交互脚本
├── update.py       # 自动更新脚本
├── deploy.sh       # 一键部署脚本
└── README.md       # 本文件
```

## 🔄 自动更新

网站内容每天自动抓取最新AI新闻并更新。

### 新闻分类
- 🔴 **重磅** - 大模型发布、重大产品更新
- 🔵 **行业** - 公司动态、市场趋势
- 🟢 **技术** - 技术突破、研究成果
- 🟡 **政策** - 监管政策、法规动态
- 🟣 **融资** - 投融资、并购动态

## 🚀 手动部署

### GitHub Pages（推荐）
1. Fork 或创建新仓库
2. 上传所有文件
3. Settings → Pages → 启用

### Vercel
```bash
npm i -g vercel
vercel --prod
```

### Netlify
直接拖拽文件夹到 [netlify.com](https://app.netlify.com/drop)
